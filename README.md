# Portfolio

Personal portfolio website developed to showcase my projects, academic activities, books, and contact information.

🌐 Live Demo: https://joaomenezes.vercel.app/

## About

This portfolio was built from scratch using HTML, CSS, and JavaScript, with a focus on simplicity, accessibility, and responsive design.

The website includes:

* About section
* Projects showcase
* Reading list
* Contact page
* Dark/Light theme support
* Responsive layout for desktop and mobile devices

## Technologies

* HTML5
* CSS3
* JavaScript
* Vercel

## Featured Projects

* **FitPlanner** — REST API for workout and fitness management built with FastAPI.
* **RustyNES** — NES emulator written in Rust.
- **Olindear** — Visual identity and UX/UI design project focused on promoting Olinda's cultural heritage.
* **C-Wordle** — Terminal-based Wordle clone developed in C.
* **ProUni-Data** — Data analysis project using Python, Pandas, and Matplotlib.
* **CPA 4All** — Arduino-based accessible controller designed for people with muscular dystrophy.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/jpedrosmenezes/portfolio.git
```

Open the project folder:

```bash
cd portfolio
```

Then open `index.html` in your browser.
